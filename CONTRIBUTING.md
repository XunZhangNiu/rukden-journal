# 📮 如何向 RUKDEN 投稿

欢迎来编乐子！RUKDEN 是一本**讽刺虚构学术期刊**，录用标准只有一条：**越离谱越好**。

## 投稿方式（二选一）

### 方式一：GitHub Discussions（推荐）

1. 点击本仓库顶部的 **Discussions** 标签页
2. 点击 **New discussion**，选择 **💡 投稿预印本**
3. 填写：

```markdown
## 作者
（你的化名）

## 分类
（矿物学 / 数学 / AI伦理 / 化学 / 味觉数学 / 感官人类学）

## 摘要
（100-200字）

## 全文
（正文内容，支持Markdown）

## 参考文献
（可选）
点击 Start discussion

方式二：直接 Pull Request

Fork 本仓库

编辑 articles.js，在数组末尾加上你的文章

提交 Pull Request

审稿标准

· ✅ 直接录用：让我笑出声
· ✅ 小修后录用：有点意思，但还能更离谱
· ❌ 拒稿：太正经了

注意

· 本刊是讽刺虚构项目
· 不要人身攻击、不要政治敏感内容
· 投稿即视为同意以 CC0 协议发布
· 只有主编（@XunZhangNiu）回复 /accept 才能录用
